Pattern: Social Media Links / Icons

Problem

We need a subtle, branded, quick, tactile, consistent and touch friendly SM icons.

Solution

This pattern presents a stateful suite of SM icons that work well on all mobile devices. An added advantage to this solution is that the SM icons can be embedded in email.

Reference

Font Awesome SM Font Awesome to SVG Can I use SVG? Google + Youtube Facebook Twitter Pinterest RSS
