Pattern: Favorite and Touch Icons

Problem

Each browser and operating system requests is own unique fav or touch icon. Many unfulfilled requests fill up the server logs with 404 requests. Mobile browsers expect touch icons to be sized properly for the home screen.

Solution

Utilize best practices to generate all required dimensions from a single branded image. This process also generates the meta tags and the manifest json / xml required by various UA/OS combinations.
